# Webpack Developer Experience Tool (Prototype)

## 📌 Overview

This project is a basic prototype of a custom plugin for webpack that improves developer experience by providing visibility into the build lifecycle.

It demonstrates how to hook into webpack’s compiler lifecycle and log build stages in the terminal.

---

## 🚀 Features

* Logs when the build starts
* Logs when the build finishes
* Demonstrates usage of webpack plugin lifecycle hooks
